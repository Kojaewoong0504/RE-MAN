## 비주얼 무드 — Bold / Paper Poster

> "변화를 시작하는 순간은, 종이 위에 크게 선언하는 것처럼 보여야 한다."

이 앱은 너드남이 처음으로 "나도 할 수 있다"는 감각을 갖게 만든다.
그 감각은 밝은 바탕, 잉크처럼 또렷한 타이포그래피, 네온 옐로우 포인트, 군더더기 없는 레이아웃에서 온다.

### 컬러 팔레트

| 용도 | 색상 | 비고 |
|---|---|---|
| Primary | `#FCF8EF` (웜 화이트) | 배경 기본 |
| Accent | `#F2E94E` (포스터 옐로우) | CTA, 강조 포인트 |
| Surface | `#F4ECDD` | 카드/패널 배경 |
| Text Primary | `#111111` | 본문 |
| Text Secondary | `#6B6558` | 보조 텍스트 |
| Success | `#4ADE80` | 미션 완료 |
| Error | `#F87171` | 실패/경고 |

### 타이포그래피

- **폰트**: Pretendard (한글), Inter (영문)
- **헤드라인**: 32px / Bold / 자간 -0.5px
- **서브헤드**: 20px / SemiBold
- **본문**: 16px / Regular / 행간 1.6
- **캡션**: 13px / Regular / `#888888`

### 기타 스타일 규칙

- 보더 라디우스: 12px (카드), 8px (버튼)
- 그림자: 최소화 — 종이 겹침 같은 면 대비 우선
- 아이콘: 선형(outline) 아이콘만 사용, filled 금지
- 이미지 필터: 유저 사진 외 스톡 이미지 사용 금지

## 레이아웃 원칙

### 기본 구조

- 모바일 우선 설계 (max-width: 480px 기준)
- 단일 컬럼 레이아웃
- 화면당 핵심 액션 1개만 — CTA는 항상 하단 고정

### 카드형 피드백 레이아웃

AI 피드백은 단계별 카드로 순차 노출한다.
한 번에 모두 보여주지 않는다 — 카드를 열어가는 경험이 중요하다.

```
[진단 카드]         ← 항상 먼저 노출
    ↓ 탭/스와이프
[개선 포인트 카드 1]
    ↓
[개선 포인트 카드 2]
    ↓
[개선 포인트 카드 3]
    ↓
[오늘의 액션 카드]  ← 가장 강조, Accent 컬러
```

각 카드는:
- 상단: 아이콘 + 라벨 (예: "개선 포인트 1")
- 중앙: 피드백 텍스트 (최대 3줄)
- 하단: 다음 카드로 넘기는 버튼

## 화면별 디자인 원칙

### 랜딩
- 헤드카피가 화면의 60% 차지
- CTA 버튼은 Accent 컬러
- 밝은 포스터형 배경, 카드 남용 금지
- 배경 이미지 없음 — 텍스트와 면 분할만으로 무게감

### 설문
- 질문 1개씩 풀스크린 노출 (한 화면에 여러 질문 금지)
- 선택지는 큰 탭 버튼 형태
- 진행 바 상단 표시 (1/3 → 2/3 → 3/3)

### 사진 업로드
- 업로드 영역 중앙 배치, 점선 테두리
- "잘 나올 필요 없어요" 카피 강조
- 가이드 예시는 작게, 방해되지 않게

### 로딩
- 밝은 배경에 잉크색 텍스트 애니메이션
- 프로그레스 바 없음 — 텍스트 전환으로 진행감 표현

### 피드백 결과
- 카드형 순차 노출
- 마지막 카드(오늘의 액션)는 Accent 배경으로 강조
- 하단 고정 CTA: "Day 1 미션 시작하기"
